## 📖 Overview

This analysis workflow aims to study the dataset of Pancreatic Adenocarcinoma (PAAD) from The Cancer Genome Atlas (TCGA) project. The analysis done in Bash and R includes: detection of driver genes with dNdScv, oncoplots and lollipop plots graphing and mutational signatures analysis for a Cohort of 165 pacients with PAAD.

## 💻 Prerequisites & Environment Setup

To ensure reproducibility of the statistical and computational analyses, we recommend using the following software:

- R (version 4.4.1), with the following packages:
  - dndscv (version 0.0.1.0)
  - maftools (version 2.22.0)
  - tidyverse (version 2.0.0)
  - reticulate (version 1.46.0)
  - SigProfilerMatrixGeneratorR
  - SigProfilerPlottingR
  - SigProfilerExtractorR
  - SigProfilerAssignmentR
- Bash

Also, we recommend using a conda environment, which can be done with the following code:
```bash
## Windows (using Anaconda Prompt)
conda create --name mutational_signatures -y
conda activate mutational_signatures
conda install python=3.10 -y

conda activate mutational_signatures
pip install SigProfilerExtractor
```
